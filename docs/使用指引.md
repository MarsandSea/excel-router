# ExcelRouter 使用指引

> Excel 智能拆分工具 —— 整个文件夹一键拆完：按部门、区域、工号等字段自动拆分，
> 保留原格式，自动打包分发。面向普通办公人员，零编程。

本指引配合软件内的**三步卡片式界面**逐步讲解。看完约 3 分钟，你就能独立完成一次拆分。

---

## 一、它能帮你做什么

你手上有一批（或一个）Excel 表，想**按某一列的取值把它拆成多个文件**——
比如一张全公司销售明细，按「区域」拆成 华北 / 华东 / 华南 / 东北 四个文件，分别发给区域负责人。

ExcelRouter 做的就是这件事，而且：

- **自动识别表头**：表头不在第一行（有大标题、分组标签）也能自动找到。
- **自动枚举取值**：你只要选「按哪一列拆」，它自己列出该列的所有取值，每个取值生成一个文件。
- **跨文件自动合并**：同一个取值出现在多个源文件里（如 1–5 月的表都有「华北」），自动合并进同一个结果文件。
- **保留原格式**：字体、颜色、边框、数字格式、合并的表头单元格都原样保留。
- **可选「到人」**：在按区域拆的基础上，再按「姓名」细分，一次产出「汇总 + 到每个人」两套结果。
- **自动打包 ZIP**：批量拆时每个分组打成一个 ZIP，方便整体转发。
- **数据不出本机**：全程本地处理，不联网、不上传。

> **一句话**：把"手动筛选 → 复制 → 新建文件 → 粘贴 → 调格式"这套重复劳动，压成三步点击。

## 二、准备：下载并打开

普通用户无需安装 Python，直接到 **[Releases](../../releases)** 下载：

| 产物 | 说明 | 建议 |
|---|---|---|
| `ExcelRouter-vX.X.X-win64.zip` | 解压后双击里面的 `ExcelRouter.exe` | ✅ **优先选这个**，启动快、极少被杀毒误报 |
| `ExcelRouter-vX.X.X.exe` | 单文件版，下载即用 | 图方便可选，个别杀毒软件可能误报 |

> 若杀毒软件报警，那是**误报**（开源工具打包的常见现象），处理办法见 [FAQ](FAQ.md#杀毒软件误报)。

## 三、三步上手（核心流程）

打开软件，界面从上到下就是三张编号卡片：**① 选表格 → ② 选字段 → ③ 开始拆分**。

### ① 选表格

两个直选按钮，按你的场景点其一：

- **📄 选一个 Excel 文件** —— 只拆单个表。
- **📁 选整个文件夹（批量拆）** —— 一次处理文件夹里所有 Excel，并启用「批量选项」（打包 ZIP、到人）。

> 也可以直接把文件/文件夹路径**粘贴**到输入框。选好后软件会自动扫描表格、推荐保存位置。

### ② 选字段

- 在 **「拆分字段」** 下拉里选**按哪一列拆**（如「区域」「所属部门」）。
- 如果下拉是空的，点 **「🔄 扫描字段」** 让它重新识别一次表头和列。
- 提示：*拆分字段的每个取值各生成一个文件——按「部门」拆 → 销售部.xlsx、财务部.xlsx…*

**批量拆时（选了文件夹）**，② 里会多出批量选项：

- ☑ **每个分组打包成 ZIP**：方便把「华北」整包直接转发。
- ☑ **同时拆到人**：在分组之外，再按「姓名」等字段把每个人单独出一个文件。勾选后设置：
  - **按哪个字段区分人**（如「姓名」）。
  - **只处理文件名含 ___ 的表**（如只对「工资」「费用」表到人，留空＝全部）。

### ③ 开始拆分

- **结果保存到**：可留空——软件会自动在数据旁边建一个「拆分结果」文件夹。想指定别处就点「浏览」。
- 点 **「▶ 开始拆分」**。运行时会显示进度条和状态；大文件（几万行）也**不会假死**，有实时进度与心跳。
- 完成后点 **「📂 打开输出文件夹」** 查看结果。

> 万一失败，下方 **「▸ 处理详情」** 日志里有具体原因（会自动展开）。

## 五、进阶与「高级设置」

② 卡片下方有 **「▸ 高级设置（一般用不到）」** 折叠区，只有在**扫描不到字段**或需要**归并取值**时才展开：

| 选项 | 什么时候用 |
|---|---|
| **表头识别**：自动 / 指定行号 / 关键词 | 自动识别不准时，改成手动指定表头在第几行 |
| **跳过值** | 拆分列里不想单独成文件的值，如「合计」「小计」「总计」（默认已含） |
| **取值归并映射（JSON）** | 把多个写法归成一个，如 `{"华北":["华北区","HB"]}` |
| **精确匹配** | 取值必须完全相等才归入（默认按包含匹配的场景关掉） |
| **跨文件合并汇总** | 关掉后，目录批量会变成「每个源文件各自拆」而非合并 |
| **保留格式** | 关掉切「快速模式」，只写值、最快，但丢格式 |
| **完成后打开输出** | 跑完自动弹出结果文件夹 |

## 六、两条要记住的限制

1. **`.xls`（老格式）转换后只保留数据、不保留格式**；`.xlsx` 正常保留。
2. **数据区的合并单元格暂不保留**（表头的合并单元格正常保留）。
3. **公式列读的是缓存值**：如果某文件的公式从没被 Excel 算过，公式列可能读成空白——
   用 Excel 打开另存一次即可。软件会提前在日志里预警这种情况。
