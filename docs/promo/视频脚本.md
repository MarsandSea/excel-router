## 分发与引流路径（重要）

**核心判断：GitHub 是「权威背书」，不是「下载入口」。** 国内普通办公人员访问 GitHub 慢、
无账号、Releases 页看不懂，视频观众更无法点击。所以：

- **下载主路径 = Gitee（码云）发布页**：国内直连快、免登录直下、保留开源背书。链接放
  **主页简介 / 置顶评论**。（需把仓库镜像/同步到 Gitee，并在 Gitee 挂 Release。）
- **信任背书 = GitHub + Gitee「开源可查」**：视频里只出现一行小字，是信号不是动作。
- **公众号 = 次级留存/更新通知，不进抖音/小红书视频**。原因：抖音、小红书都压制往微信生态
  导流（小红书尤其严），视频/主页出现「公众号/微信」易限流。公众号引导只放**视频号 / B站 /
  知乎**版本，或用户下载后在软件/Gitee 页引导。
- **抖音/小红书 CTA 机制**：不硬塞二维码（会限流），用「品牌词 ExcelRouter + 主页/评论区领取」。
  真正的 Gitee 链接落在简介/置顶评论。
- **二维码**：只做**视频号 / B站**版本（这两个平台可较自由展示外链/二维码）。
- **AI 学习号与本工具分开**：作者规划中的「AI 学习记录」公众号是另一内容赛道，不要和本工具的
  下载漏斗混用，避免稀释两边定位。

> 收尾帧文案（hook_15s.html 拍4）已据此定为：「👆 主页领取 · 完全免费」+「全网同名
> ExcelRouter · 开源免费可查」，不再写「GitHub 搜」这种视频里点不动、国内又慢的引导。

## 版本 A｜真实操作（约 20 秒，录屏）

**录制前准备**：打开软件、准备好 `examples/` 样本文件夹、把窗口调到竖屏友好的比例（或录制后裁剪）。

| 时间 | 你的操作（录屏） | 叠加字幕 |
|---|---|---|
| 0–2s | **直接给结果**：先闪一眼输出里的 `华北/到人/张伟.xlsx 王芳.xlsx…`（先甜后苦，留住人） | **「整个文件夹 → 自动拆到每个人」** |
| 2–6s | 回到软件，点 **📁 选整个文件夹**，选 `examples/` | **「① 选文件夹」** |
| 6–11s | 「拆分字段」选 **区域**，勾 **「☑ 同时拆到人」**、区分人选 **姓名**（顺手勾「每组打包 ZIP」） | **「② 选字段 + 勾一下到人」** |
| 11–15s | 点 **▶ 开始拆分**，进度条跑完 | **「③ 开始拆分」** |
| 15–20s | 点 **📂 打开输出文件夹**，进 `华北/`：**汇总 xlsx + 到人文件夹里每人一份** + ZIP | **「区域汇总 + 每人一份 · 一次全出 · 免费开源」** |

## 录制与导出

### A. 录动画（版本 B）
1. 双击 `docs/promo/hook_15s.html`，浏览器全屏播放。
2. 用录屏工具对准中间手机框录**一个完整循环（16 秒）**：
   - **最省事**：`Win + G`（Windows 自带 Xbox Game Bar）→ 录制。
   - **要透明/精细**：[ScreenToGif](https://www.screentogif.com/)（免费，可直接导 GIF/MP4，能逐帧删改）。
   - **要高质量**：OBS Studio（免费）。
3. 裁掉多余边缘，导出 9:16。

### B. 录软件（版本 A）
- 同样用 ScreenToGif / OBS 录软件窗口的真实操作，按上面的分镜走一遍即可。

### C. 导出规格建议
- 尺寸 **1080×1920（9:16）**；帧率 30fps；时长 **15–25 秒**（越短完播率越高）。
- 首帧就是钩子（痛点或结果），不要有 logo 开场空镜。

### D. 从长录像自动剪精华（你的思路）
录一段较长的完整演示，再用 **ffmpeg** 按时间点截取拼接（纯 CPU，几秒完成，不吃显卡）：
```bash
# 例：从 full.mp4 截取 3 个精华片段并快速拼接
ffmpeg -i full.mp4 -ss 00:00:02 -to 00:00:06 -c copy a.mp4
ffmpeg -i full.mp4 -ss 00:00:14 -to 00:00:18 -c copy b.mp4
ffmpeg -i full.mp4 -ss 00:00:30 -to 00:00:34 -c copy c.mp4
(echo file a.mp4 & echo file b.mp4 & echo file c.mp4) > list.txt
ffmpeg -f concat -safe 0 -i list.txt -c copy out.mp4
```
> 需要「按说话停顿/画面变化自动找精华点」的话，可以再上一层用 auto-editor（基于静音检测自动去空段），
> 也是 CPU 工具。需要我给你配好命令随时说。

## 算力说明（放心用）

- **做动画** = 写 HTML/CSS 代码，浏览器播放，**几乎零算力，无需显卡、无需联网**。
- **导出成片** = 屏幕录制（免费工具）或 ffmpeg 截帧拼接（CPU 几秒），**都不吃显卡**。
- 全程不涉及 AI 视频生成那种重算力，你现有机器完全够。
